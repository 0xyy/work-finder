## Run Locally or on scan the QR code with installed expo

Clone the project

```bash
  git clone https://github.com/0xyy/swift-shop.git
```

Go to the project directory and install dependencies

```bash
  npm install
```

Start the server in development mode

```bash
  npm start
```
